# JokeBot-FastAPI-MongoDB
End-to-end LLM application demonstrating API development (FastAPI), NoSQL database integration (MongoDB), and rapid UI prototyping (Streamlit). Implements custom history retrieval and user session management.

🃏 Joke Bot AI
An end-to-end Chatbot built with FastAPI, Streamlit, and LangChain.

🛠️ Tech Stack
Frontend: Streamlit (Custom Chat UI)

Backend: FastAPI (Python)

LLM: Groq (Llama3/GPT-OSS) via LangChain

Database: MongoDB (Chat History Persistence)

🚀 Features
Persistent History: Messages are stored in MongoDB and retrieved via a GET endpoint.

User Authentication: Simple ID-based login system.

Real-time Streaming: Fast joke generation using Groq.
